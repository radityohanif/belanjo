=================================
Tutorial Install Aplikasi Belanjo
=================================

1. Download Folder Belanjo
2. Buka aplikasi XAMPP
3. Jalankan module Apache & MySQL dengan menekan tombol start pada kolom Actions (tunggu sampai berwarna hijau)
4. Buka aplikasi web browsermu (Google Chrome, Microsoft Edge, dll) lalu ketikan alamat url berikut "http://localhost/phpmyadmin/" dan tekan enter
5. Pilih menu "impor" pilih "Choose File" lalu arahkan ke file "database.sql" yang ada pada folder ini, setelah itu tekan tombol kirim (terletak pada bagian bawah) untuk memulai proses pembuatan database
6. Pindahkan folder Belanjo ini kedalam direktori "C:\xampp\htdocs"
7. Silahkan kunjungi alamat url berikut "http://localhost/belanjo/public/" dengan menggunakan aplikasi web browsermu untuk menjalankan aplikasi belanjo
